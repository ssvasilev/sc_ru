# STAR

**[STAR (Script To Apply Russian)](https://github.com/ssvasilev/STAR/)** — это Windows batch-скрипт для автоматической установки и обновления русской локализации в **Star Citizen**

Программа создана для русскоязычного сообщества и распространяется бесплатно, без рекламы и коммерческих целей. Она не требует установки, обладает минималистичным и полностью автоматизированным функционалом, а также практически не требует участия пользователя. Небольшой размер обеспечивает высокое быстродействие, а открытый исходный код позволяют убедиться в отсутствии вредоносных действий.
При запуске программа автоматически определяет и устанавливает актуальные файлы локализации, используя релизы с GitHub ([ссылка](https://github.com/n1ghter/StarCitizenRu)).

Файлы локализации содержат текстовые строки, используемые в игре. Использование программы и файлов русской локализации не оказывают негативного влияния на работоспособность игры.

Программа предоставляется «как есть» (AS IS), без каких-либо гарантий. Использование осуществляется на ваш страх и риск.

## 📦 Быстрый старт
- Вы можете скачать файл в удобное место, рекомендуется разместить его рядом с RSI Launcher.exe. Тогда при запуске путь к лаунчеру определится автоматически и потребуется указать путь только к каталогу с игрой.
- Для поддержания актуальности локализации рекомендуется каждый раз запускать игру через эту программу — она автоматически проверит наличие обновлений, при необходимости обновит локализацию и затем запустит игру.

## ✨ Возможности программы

- Поддержка веток LIVE и PTU* (основной и тестовой версий игры)
- Автоматическая проверка и обновление локализации
- Защита от установки несовместимых версий
- Автоматическое подключение локализации через `user.cfg`, с созданием резервной копии.
- Запуск RSI Launcher после выполнения всех операций

*PTU - специальная тестовая версия игры, предоставляемая разработчиками некоторым игрокам. При первом запуске, программа спросит, настраивать ли путь к каталогу с PTU. Если у вас нет доступа к PTU, нажмите "n".

## ⚙️ Как работает программа

1. Определяются пути из файла конфигурации программы:
   - к `RSI Launcher.exe`
   - к папкам с версиями игры **LIVE** и **PTU** (если настроена)
2. Проверяет установленную версию локализации (если она есть)
3. Получает актуальные данные из GitHub
4. Сравнивает версии и при необходимости загружает обновление
5. Проверяет совместимость файлов
7. Устанавливает локализацию в папку игры
8. Делает резервную копию файла `user.cfg` и обновляет его.
9. Запускает лаунчер игры
