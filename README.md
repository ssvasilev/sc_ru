# STAR

**STAR (Script To Apply Russian)** — это Windows batch-скрипт для автоматической установки и обновления русской локализации в **Star Citizen**.

Скрипт упрощает процесс установки локализации, автоматически определяет версии и устанавливает подходящие файлы из GitHub-релизов.

---

## ✨ Возможности

- Поддержка веток **LIVE** и **PTU**
- Автоматическое определение путей к игре и лаунчеру
- Проверка уже установленной локализации
- Получение актуальных версий из GitHub
- Скачивание и проверка архивов локализации
- Защита от установки несовместимых версий
- Автоматическое обновление `user.cfg`
- Запуск RSI Launcher после установки

---

## ⚙️ Как это работает

1. Определяются пути к:
   - `RSI Launcher.exe`
   - папкам игры (**LIVE**, **PTU**)
2. Проверяется текущая установленная версия локализации
3. Получаются последние релизы с GitHub
4. Сравниваются версии
5. При необходимости скачивается архив
6. Выполняется проверка:
   - структуры файлов
   - соответствия ветке
   - корректности версии
7. Файлы копируются в папку игры
8. Обновляется `user.cfg`
9. Запускается лаунчер

---

## 🖥 Требования

- Windows
- PowerShell
- Установленный **Star Citizen**
